# Explorer

## Installation

Clone with `--recurse-submodules`-flag to automatically initialize and update each submodule.

```
git clone --recurse-submodules git@github.com:teco-kit/explorer.git

```

## Docker Environment

```
sudo docker-compose build
docker-compose up

```

* explorer-frontend:
* explorer-backend:
